# 删除Key与值    

* TCP
命令: `delete <key>\n`
~~~shell
delete hello\n
~~~

* HTTP  
>⚠⚠⚠在添加和修改数据时，建议遵循HTTP协议，并使用专业客户端进行后期提交，以避免传输编码导致数据不一致。这只是一个示范。  

<table>
    <tr>
        <td>请求参数</td>
        <td>参数值</td>
    </tr>
    <tr>
        <td>cmd</td>
        <td>delete</td>
    </tr>
    <tr>
        <td>key</td>
        <td>key名称</td>
    </tr> 
    <tr>
        <td>token</td>
        <td>token值</td>
    </tr>
</table> 

~~~shell
http://127.0.0.1:7721/data/?cmd=delete&key=hello&token=8a31af73dfd56ea863b1281289e76e32
~~~
<br>

响应结果例子:
~~~shell
{
    "Cmd": "delete",
    "Status": 200,
    "Results": "SUCCESS",
    "Key": "hello",
    "Data": null,
    "ResponseHost": "127.0.0.1:7721",
    "KeyCreateTime": "0001-01-01T00:00:00Z"
}
~~~
