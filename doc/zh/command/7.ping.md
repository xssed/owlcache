# Ping 命令  

>该命令支持的版本 >= 0.4.2

* TCP
命令: `ping <string>\n` 或者 `ping\n`  

示例1:  

~~~shell
ping hello\n
~~~

示例1 Result:  
~~~shell
hello
~~~

<br>
示例2:

~~~shell
ping\n
~~~

示例2 Result:  
~~~shell
PONG
~~~
<br>

* HTTP
>⚠⚠⚠在添加和修改数据时，建议遵循HTTP协议，并使用专业客户端进行后期提交，以避免传输编码导致数据不一致。这只是一个示范。  

<table>
    <tr>
        <td>请求参数</td>
        <td>参数值</td>
    </tr>
    <tr>
        <td>cmd</td>
        <td>ping</td>
    </tr>
    <tr>
        <td>响应内容(不是必须项)</td>
        <td>想要ping命令返回的约定字符串</td>
    </tr> 
</table> 

示例1:  

~~~shell
http://127.0.0.1:7721/data/?cmd=ping
~~~

示例1响应结果:  
~~~shell  
PONG
~~~  


示例2:  
~~~shell  
http://127.0.0.1:7721/data/?cmd=ping&valuedata=hello
~~~  

示例2响应结果:  
~~~shell  
hello
~~~  

<br>
