# 检查一个Key是否存在
* TCP
命令: `exist <key>\n`
~~~shell
exist hello\n
~~~
* HTTP

<table>
    <tr>
        <td>请求参数</td>
        <td>参数值</td>
    </tr>
    <tr>
        <td>cmd</td>
        <td>exist</td>
    </tr>
    <tr>
        <td>key</td>
        <td>key名称</td>
    </tr> 
</table>  

~~~shell  
http://127.0.0.1:7721/data/?cmd=exist&key=hello1  
~~~
<br>

响应结果例子:
~~~shell
{
    "Cmd": "exist",
    "Status": 200,
    "Results": "SUCCESS",
    "Key": "hello1",
    "Data": null,
    "ResponseHost": "127.0.0.1:7721",
    "KeyCreateTime": "0001-01-01T00:00:00Z"
}
~~~
