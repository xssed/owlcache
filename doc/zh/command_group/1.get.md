# 获取Key值

* HTTP
注意:HTTP获取数据不用密码验证，只有其他更改数据的操作需要验证身份。

<table>
    <tr>
        <td>请求参数</td>
        <td>参数值</td>
    </tr>
    <tr>
        <td>cmd</td>
        <td>get</td>
    </tr>
    <tr>
        <td>key</td>
        <td>key名称</td>
    </tr> 
</table> 
 
~~~shell
http://127.0.0.1:7721/group_data/?cmd=get&key=hello
~~~
