# Check if a Key exists
* TCP
command: `exist <key>\n`
~~~shell
exist hello\n
~~~
* HTTP

<table>
    <tr>
        <td>parameter name</td>
        <td>parameter value</td>
    </tr>
    <tr>
        <td>cmd</td>
        <td>exist</td>
    </tr>
    <tr>
        <td>key</td>
        <td>key name</td>
    </tr> 
</table>  

~~~shell  
http://127.0.0.1:7721/data/?cmd=exist&key=hello1  
~~~
<br>

Response result example:
~~~shell
{
    "Cmd": "exist",
    "Status": 200,
    "Results": "SUCCESS",
    "Key": "hello1",
    "Data": null,
    "ResponseHost": "127.0.0.1:7721",
    "KeyCreateTime": "0001-01-01T00:00:00Z"
}
~~~
