# Get the Key value
* TCP
command: `get <key>\n`
~~~shell
get hello\n
~~~

* HTTP
Note: HTTP access data is not verified by password, only other operations that change data require authentication.

<table>
    <tr>
        <td>parameter name</td>
        <td>parameter value</td>
    </tr>
    <tr>
        <td>cmd</td>
        <td>get</td>
    </tr>
    <tr>
        <td>key</td>
        <td>key name</td>
    </tr> 
</table> 
 
~~~shell
http://127.0.0.1:7721/data/?cmd=get&key=hello
~~~
<br>

Response result example:
~~~shell
world
~~~

# Get the Key value info
* TCP
command: `get <key> info\n`
~~~shell
get hello info\n
~~~

* HTTP
Note: HTTP access data is not verified by password, only other operations that change data require authentication.

<table>
    <tr>
        <td>parameter name</td>
        <td>parameter value</td>
    </tr>
    <tr>
        <td>cmd</td>
        <td>get</td>
    </tr>
    <tr>
        <td>key</td>
        <td>key name</td>
    </tr> 
    <tr>
        <td>valuedata</td>
        <td>info</td>
    </tr>
</table> 
 
~~~shell
http://127.0.0.1:7721/data/?cmd=get&key=hello&valuedata=info
~~~
<br>

Response result example:
~~~shell
{
    "Cmd": "get",
    "Status": 200,
    "Results": "SUCCESS",
    "Key": "hello",
    "Data": "",
    "ResponseHost": "127.0.0.1:7721",
    "KeyCreateTime": "2021-11-09T14:12:36.8431596+08:00"
}
~~~
